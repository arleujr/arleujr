<details open>
<summary><strong>Core Stack and Practical Application</strong></summary>
<br>

Here is the real-world context of how I apply my tools and studies across active projects and experiments.

### Data Engineering and Databricks Ecosystem
* **Medallion Architecture and Data Quality** Applied studies in data layering (Bronze, Silver, and Gold) using PySpark and Delta Lake, focusing on rebalancing the Silver layer to mitigate intermediate structural issues like schema drift.
* **Pipeline Monitoring** Practical implementation of structured validation systems (try/catch blocks, schema validations, and structured logging) aimed at tracking execution failures and ensuring strict SLA deliveries.

### Python, FastAPI and Asynchronous Processing
* **Ingestion Pipelines** Development of event-driven microservices and asynchronous workers in Python built for continuous ingestion of agricultural field telemetry data streams.
* **Resilience Mechanisms** Implementation of defensive code patterns such as Exponential Backoff with Jitter to ensure application stability against infrastructure down times or database connection drops.
* **Reference Repository** [agrisentry-core](https://github.com/arleujr/agrisentry-core) (Python, FastAPI, SQLAlchemy, AsyncIO, Pytest).

### Databases and Storage Performance
* **Concurrency Mitigation** Practical use of database transaction mechanics like `FOR UPDATE SKIP LOCKED` for secure, race-condition-free data isolation across multiple concurrent background workers.
* **Query Optimization** Eliminating data performance bottlenecks (such as the N+1 query problem) utilizing analytical Window Functions and optimized batch ingestion processing.
* **SGBDs in Use** PostgreSQL, Supabase/TimescaleDB, SQLite (dialect-isolated automated testing setups), NoSQL (MongoDB).

### Frontend, Products and AI Analytics
* **Reactive Interfaces** Building Vue.js Single-Page Applications (SPAs) integrated with backend APIs, specializing in real-time simulation matrices and continuous hardware sensor status logs.
* **Data Visualisation (Power BI and Web)** Transforming raw server datasets and data frames into interactive operational dashboards, ensuring high data readability for business stakeholders.
* **Reference Repository** [agrisentry-dashboard](https://github.com/arleujr/agrisentry-dashboard) (Vue 3, TailwindCSS, Vite).

### Ingestion Gateway (Infrastructure)
* **High-Speed Ingestion** Constructing lightweight networking blocks in Rust to receive and route simultaneous multi-protocol streams (HTTP/MQTT) without non-blocking database IO bottlenecks.
* **Reference Repository** [agrisentry-iot-gateway](https://github.com/arleujr/agrisentry-iot-gateway) (Rust, Actix-Web, SQLx, Rumqttc).
</details>
